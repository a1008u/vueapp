# mymarkdown

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```

---
## Firebaseについて

### Firebaseの設定
```
npm install -g firebase-tools
firebase login
firebase init
```
### Firebaseへのデプロイ
```
yarn run build
firebase deploy
```
